The name of the commit status check displayed in the pull request. The commit's status check name, shown on the pull request, is typically configured with the GitHub plugin. See the 'Advanced' settings of the job's GitHub project option. When that setting is insufficient, a custom name can be specified here to help identify the particular commit status in use. This setting affects the standard GitHub project value by appending custom text to the original name. The final name of the check is created as:
  • If the job's "GitHub project" setting has a configured "display name"...
    • Without a custom name: displayName
    • With a custom name: displayName-customName
  • If the GitHub project does not have a configured "display name"...
    • Without a custom name: jobName
    • With a custom name: customName
Note, if using a custom name, ensure that any other corresponding status updates use the same name to ensure they update the same commit status.
